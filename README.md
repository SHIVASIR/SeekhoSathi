# 📘 SeekhoSathi – Learning ka Companion

SeekhoSathi is a **clean, user-friendly, step-by-step learning web application** designed for students who want to **understand concepts first and practice later**.

The platform focuses on **clarity, simplicity, and structured learning**, inspired by modern EdTech products.

---

## 🌟 What is SeekhoSathi  ?

SeekhoSathi helps learners:
- Understand any topic in **simple English**
- Learn the same concept in **Hinglish (Hindi + English)**
- Follow a **step-by-step learning flow**
- Practice concepts using **topic-based quizzes**

> ✨ *Learning should feel simple, not overwhelming.*

---

## 🔁 Learning Flow

Landing Page (About the product)
↓
Login / Get Started
↓
Topic Selection
↓
Learn (Definition + Explanation)
↓
Quiz Practice

## ▶️ How to Run the Project

1. Download or clone the repository  
2. Open the project folder in **VS Code**  
3. Open `home.html` in any browser  
4. Follow the flow and start learning 🚀  

> No backend or installation required.
